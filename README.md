# The Loudening

A Tipping Point editorial dispatch on ocean noise and whale silence.

## What this is

A scrollytelling landing page that lets you hear what we are doing to the ocean — using **real sperm whale recordings** from the Dominica Sperm Whale Project (DSWP), combined with documented research on ambient noise pollution.

## How to view

Open `index.html` in any modern browser. Headphones recommended for the listening sections.

## How to publish

This is a fully static site. Drop the entire folder onto:
- **GitHub Pages** — push to a repo, enable Pages, done
- **Netlify** — drag and drop the folder onto netlify.app
- **Vercel** — `vercel deploy` from the folder
- Any web host

No build step, no dependencies. Three files: `index.html` and the five WAVs in `audio/`.

## Audio credits

All whale recordings are real sperm whale codas from the Dominica Sperm Whale Project archive, hosted on Hugging Face by Project CETI:

- Dataset: https://huggingface.co/datasets/orrp/DSWP
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
- Recorded: 2005–2018 off the coast of Dominica
- Method: boat-based hydrophones and animal-borne acoustic tags
- Citation: Paradise et al., "Towards A Translative Model of Sperm Whale Vocalization" (NeurIPS 2025)

## Coda files included

| File | Type | Clicks | Duration | Notes |
|---|---|---|---|---|
| `10.wav` | 1+1+3 | 5 | 2.20s | Eastern Caribbean signature dialect |
| `1000.wav` | 6+1 | 7 | 2.50s | Long-pause coda |
| `1004.wav` | Complex | 17 | 2.42s | Multiple whales overlapping |
| `1017.wav` | Exchange | 13 | 1.57s | Rapid back-and-forth |
| `1018.wav` | Chorus | 18 | 1.68s | Group vocalization |

Coda type classifications were generated by Teager-Kaiser energy operator click detection (the same approach used in CETI's published Coda Detector, Gubnitsky et al., Scientific Reports 2025) followed by inter-click interval analysis.

## Tech notes

- Pure HTML/CSS/JS, no frameworks
- Web Audio API for layered playback (real coda + synthesized ship noise)
- Intersection Observer for scroll-triggered reveals
- Fonts: Fraunces (display) + Inter Tight (body), both via Google Fonts CDN
- Mobile responsive
- No tracking, no analytics, no cookies
